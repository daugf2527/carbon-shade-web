
### 需要加到白名单的

| 路径 | 原因 | 明天在 Windows 上做什么 |
|------|------|----------------------|
| **`!.claude/memory/`** | 项目级记忆系统——你在 Windows 上建的多层记忆文件全被 `.claude/*` 吃掉了，git 不跟踪 | 加白名单 → `git add .claude/memory/` → commit → push |
| **`.claude/mcp.json`** | MCP 自定义配置——如果 Windows 上配了非插件市场的 MCP server，这边拿不到 | 如果文件存在就加白名单 `!.claude/mcp.json`；如果 MCP 全在 `~/.claude.json` 里则不需要 |

### notify.mjs 跨平台兼容

当前硬编码 `powershell.exe` + `BurntToast`（Windows 专属）。Termux 上 `termux-notification` 已安装且可用。需要加平台检测：

```js
// 伪代码
if (process.platform === 'android') {
  execSync(`termux-notification --title 'Claude Code' --content '${msg}'`);
} else {
  // 现有 Windows 逻辑
}
```

`termux-notification` 实测可用，不依赖 `termux-api` 包之外的任何东西。
