# DNF Combat Systems Master Specification

日期：2026-04-29

## 文档目的

本规格书对 DNF 1:1 复刻所需的全部战斗系统进行归类与优先级编排。所有系统划分为五大类：战斗内核系统、角色构筑影响战斗系统、装备与流派影响战斗系统、副本与 Raid 影响战斗系统、服务端裁决与验证系统。每类内部再按 P0（基础）→ P1（核心）→ P2（重要）→ P3（增强）→ P4（支撑/打磨）排列优先级。

## 一、战斗内核系统

战斗内核系统是战斗框架的基础，后续所有上层系统均依赖内核的正确性与完整性。P0 为"缺了就跑不起来"，P1 为"缺了手感就不对"，P2 为"缺了体验就不完整"。

## 七、实施建议路线

1. **第一阶段（P0）**：完成 13 个基础系统 → 战斗内核能跑，角色能升级，装备能穿，副本能进出，服务端能校验
2. **第二阶段（P1）**：完成 24 个核心系统 → 手感正确，build 成型，boss 有阶段，PvP 规则分离
3. **第三阶段（P2）**：完成 25 个重要系统 → 体验完整，深度足够，Raid 可玩
4. **第四阶段（P3+P4）**：完成 16 个增强/支撑系统 → 打磨品质，优化体验
